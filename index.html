Linguagem de programação: o que é?

(Voltar)

Linguagem de programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas.

Por que o HTML não é uma linguagem de programação?

(Voltar)

O HTML significa Hyper Text Markup Language (Linguagem de Marcação de HiperTexto), embora tenha linguagem no nome, ele não é uma linguagem de programação. O HTML é um arquivo de marcação, os textos são delimitados por nomes que o navegador consegue interpretar, cada nome destes delimitadores possue uma função e nome especifíco.

Como é uma estrutura básica de um HTML?

(Voltar)

A estrutura básica do HTML5 é parecida com as estrutra do XHTML e do HTML 4, contendo elementos HEAD e BODY. O DOCTYPE define qual tipo de arquivo você está escrevendo, ela é responsável por informar ao navegador qual é este tipo. É o primeiro código a ser adicionado em uma página.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.
